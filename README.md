Pigeon4Mziuri

Pigeon
Pigeon is a Java-based desktop mail client that allows you to easily manage and browse your email.

Features
-User registration and login
-Password recovery (Forgot Password):
Personal recovery password
via Gmail
-Inbox management
-Spam and blocked emails list
-Saving drafts and sent emails
-Local storage of user settings (serialization)
-Custom labels (Custom Labels) — add categories for emails
-User profile — multi-user support
-Profile deletion
-FAQ scene
-Most Common Recipients — quick access to your most frequent contacts
-Gmail integration — synchronization with Gmail accounts
-Email search — simple email search function
-Aesthetic UI — attractive and user-friendly design

Installation and Running (using .jar file)
Make sure the following components are properly set on your computer:
-Java JDK

If opening the project from IntelliJ IDEA:
Make sure that:
-The project is loaded with Maven (pom.xml file)
-Java JDK is Properly setup

If opening the exe file:
-Set up the "Pigeon.NestSetup.exe"
